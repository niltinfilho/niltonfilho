Como enviar um formulário direto pro WhatsApp usando JavaScript puro

Para compreender esse artigo totalmente será necessário um conhecimento básico de programação, especialmente em JavaScript. Ainda assim, vou esplicar detalhe por detalhe de toda lógica por trás dessa funcionalidade.

Essa será uma ótima opção para quem está iniciando no universo front-end e deseja adicionar uma interação com o usuário muito legal no seu site, simpes, fácil e funcional.

Estrutura do formulário no HTML

Para iniciarmos essa jornada será necessário fazer toda a estruturação do formulário no HTML, existem diversas formas de fazer isso, então o exemplo que vou dar será apenas uma opção, mas indepedente da forma que estiver estruturado seu formulário ele vai funcionar, pois o mais importante será o atributo "id" nos inputs/textareas.

Criando os inputs/textareas

Nesse caso eu adicionei 2 inputs, de nome e e-mail, e 1 textarea, para o usuário poder adicionar uma mensagem. Como eu havia dito, você pode fazer a estrutura do seu formulário da maneira que preferir, o importante aqui será usarmos o atributo "id", pois vamos precisar dele quanto estivermos fazendo a lógica no JavaScript, que é o que vamos fazer agora.

Adicionando o atributo ID

Pronto! Por enquanto vamos deixar o HTML de lado e mais tarde voltamos nele. Agora vamos focar na lógica do JavaScript.

Criando a função sendWhatsapp()

Para iniciarmos no JavaScript, vamos precisar de uma função, que nesse caso eu escolhi o nome "sendWhatsapp", já que vamos enviar o formulário para o WhatsApp. E, já fica a dica, sempre que for usar variáveis ou funções, procure um nome que faça sentido de acordo com a utilização daquele componente, e por uma questão de convenção, coloque o nome em inglês, mas não é obrigatório.

Pegando as informações dos campos do formulário

Vou dividir essa etapa em 3 partes, que vão se repetir para cada campo dentro do formulário.

1. Criando a variável

Primeiro precisamos criar a variável que irá armazenar a informação do campo do formulário. Para isso, use a palavra reservada "let", que serve para declarar variáveis com escopo limitado, neste caso, ao escopo da função.

Ela será responsável por armazenar o conteúdo que o usuário digitou no formulário para usarmos mais para frente.

2. Encontrando o campo do formulário

Chegou a hora de usar o atribudo ID que colocamos nos inputs/textareas do formulário. Para "encontrar" o campo que precisamos, vamos usar o "document.getElementById()", e dentro dos parenteses, usando aspas duplas, vamos colocar o ID do campo.

Esse "document" quer dizer que o JavaScript irá fazer uma busca por todos os elementos dentro do documento HTML, envolvendo toda a árvore de tags HTML. Aconselho fazer algumas pesquisas sobre DOM (Document Object Model) para entender melhor como tudo isso funciona no JavaScript.

Já o "getElementById" vai "pegar" o elemento que estiver usando o ID indicado dentro dos parenteses.

3. Pegando o valor digitado no campo

Por fim, após encontrarmos o campo desejado, precisamos buscar o valor que o usuário digitou. Para isso vamos usar a propriedade "value".

Na prática ela irá acessar a resposta do usuário e após isso podemos guardar essa informação na nossa variável.

Conclusão: O melhor sistema é o que te permite começar agora

Como eu disse no começo: se você está começando, use o que tem em mãos. Não deixe de aprender a programar porque ouviu alguém dizer que precisa de Linux ou que Mac é “o melhor”.

Com o tempo, você mesmo vai entender as diferenças e, se for necessário, poderá mudar de sistema operacional conforme as demandas dos projetos que for enfrentando.
